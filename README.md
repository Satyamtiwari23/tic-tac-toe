# 🎮 Tic Tac Toe

A modern and responsive Tic Tac Toe game built using **HTML, CSS, and JavaScript**. The project features a clean glassmorphism-inspired UI, smooth animations, hover previews, live score tracking, and responsive design for an enjoyable user experience.

## ✨ Features

* 🎯 Two-player gameplay (Player X vs Player O)
* 👀 Hover preview before placing a move
* 🏆 Automatic winner detection
* ✨ Winning animation with highlighted cells
* 🤝 Draw detection
* 🔄 Reset Board
* 📊 Live Scoreboard (X Wins, O Wins & Draws)
* ♻️ Reset Scores
* 📱 Responsive design for desktop and mobile devices
* 🎨 Modern UI with smooth animations

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (ES6)
